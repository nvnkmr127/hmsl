# HMS Installer Icons & Brand Assets

The image files in this folder are placeholders used to satisfy NSIS compiler dependencies during compilation. 

Before building the final production release, please replace these with your actual brand assets:

1. **icon.png / install.ico**: 
   - Replace with your actual 256x256 PNG icon and convert it to a standard Windows `.ico` format.
2. **sidebar.bmp**: 
   - Size: 164x314 pixels.
   - Format: 24-bit uncompressed BMP.
3. **header.bmp**: 
   - Size: 150x57 pixels.
   - Format: 24-bit uncompressed BMP.
